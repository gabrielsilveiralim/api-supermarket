## API Supermarket

API desenvolvida para gerenciamento de produtos de supermercado.

## Funcionalidades

* Cadastro de produtos
* Listagem de produtos
* Atualização de dados
* Remoção de produtos

## 🔗 Endpoints

| Método | Rota          | Descrição       |
| ------ | ------------- | --------------- |
| GET    | /products     | Listar produtos |
| GET    | /products/:id | Buscar por ID   |
| POST   | /products     | Criar produto   |
| PUT    | /products/:id | Atualizar       |
| DELETE | /products/:id | Remover         |

## ⚙️ Como executar

```bash
git clone https://github.com/gabrielsilveiralim/api-supermarket.git
cd api-supermarket
```
